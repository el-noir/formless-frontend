# Embed Developer Docs (A3.5)

This bundle is the install-and-validate path for ZeroFill embeds.

## Fast Path (Target: <15 minutes)

1. Copy the bubble snippet from your form Share tab.
2. Paste it into your site before `</body>`.
3. Add your site domain in ZeroFill allowed domains.
4. Load your site, click widget, verify chat starts.
5. Confirm events in dashboard:
   - `widget_loaded`
   - `widget_opened`
   - `conversation_started_from_embed`
